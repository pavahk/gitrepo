Introduction

Wordpress is a tool that is widely known and utilized across the internet. It is a platform that gives you the tools to create your own website, from the domain name to the design of the site itself. The cost ranges from $50 a year to $3,800 a year, though their baseline professional plan is only $300 annually. As far as the actual features of the site go, I would say they are all fairly standard for the type of site it is. I would have to say the most impressive standalone feature of the site would have to be its site editor. This technology allows for the end-user to effortlessly edit and add HTML and CSS elements on their site. It and other platforms like it have made learning HTML all but a thing of the past in the modern era, though obviously it is still useful to understand the fundamentals of web design. Here we can see the basic layout of the initial stage of the site editor. It provides an overview of the site design process. If you click on the main block in the last page it brings you to the site as it presents to the end-user. This allows the designer to in real time see how the page changes to edits. This is the most important feature of the site, to me. Since this is just an overview of “one feature” I won’t go into all the dozens of various options one can engage with on the site editor. I will, however, go over the basic principles of the site editor. The site we see in the screenshot is the default blog that Wordpress will generate for you. There are other templates, but this is the default one. You can transform and edit the various pieces of the site by clicking onto the various blocks and interacting with the elements presented. It is a fairly intuitive process, one that is facilitated by iconographic design elements as well as pop-ups on the elements that describe their purpose. Personally if I were designing a page with Wordpress I would likely start with another template, but as this is the default it is what I want to explain the use of the site with.

Pages

If you want to add a page to the site you do the following: First, from the site editor page, hit the wordpress logo in the top left Then, navigate to “Pages” Then, click “Add New Page” This will allow you to create pages for individual pieces of content for your site.

Posts

You can also create “Posts” which are less permanent than “Pages” by returning to the home screen and clicking on "Posts". Then you can click “Add New Post” Here you can edit the post like you would edit something on the pages of the site. This is good for editorial content that needs to frequently be posted. These are all the basic functions of WordPress. It is a fairly intuitive system that tries to explain itself and tutorialize along the way. It can be a useful tool for both creating a site for actual enterprise or personal use, or for just mocking up a site for a proposal.
